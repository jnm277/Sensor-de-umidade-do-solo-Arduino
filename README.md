# Sensor-de-umidade-do-solo-Arduino
Projeto com circuito e código para conectar um sensor de umidade do solo a um arduino uno r3

Ilustração do circuito pronto:
<img width="1408" height="636" alt="Image" src="https://github.com/user-attachments/assets/2a03bab3-5580-49bb-970b-16e261c3a8e5" />
Conexões dos pinos:
> 5v => pino "Vcc" do sensor; <br>
> GND => protoboard e pino "gnd" do sensor;<br>
> A0 => pino "signal" do sensor;<br>
> Pino digital 2 => LED vermelho;<br>
> Pino digital 3 => LED amarelo;<br>
> Pino digital 4 => LED verde.<br>

Valor dos resitores:<br>
> 10k ohm.<br>
